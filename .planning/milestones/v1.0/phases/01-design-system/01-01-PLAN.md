<design_brief>
**Project**: Silver Sycamore Staff Hub — internal knowledge management platform for venue staff

**Aesthetic direction**: Archive/editorial — think museum catalog, architectural publication, or high-end reference book. Clean, authoritative, professional.

**Audience**: Venue staff (event coordinators, cafe staff, operations) accessing operational procedures daily. Needs to feel trustworthy, easy to read, and elevated above typical internal tools.

**Tone**: Refined minimalism with editorial character. Not playful, not corporate bland. Distinctive without being distracting.

**Typography requirements**:
- Display font: Distinctive serif with character (bold weights 600-700 for impact)
- Body font: Highly legible sans-serif that pairs well
- AVOID: Inter, Roboto, Arial, system fonts, or any generic choices

**Color requirements**:
- Dominant: Near-black text on light backgrounds
- Clear hierarchy: primary, secondary, muted text levels
- AVOID: Purple gradients, pastel palettes, generic SaaS blue
- Commit to palette with character

**Differentiator**: Should feel like opening a well-designed reference book, not a generic web app.

**Technical constraints**:
- Tailwind CSS v4
- CSS custom properties (variables)
- Google Fonts (must be available there)
- Existing structure: --space-xs through --space-2xl (keep this)
</design_brief>
