  <action>
Create Button component with variants:

**Props:**
- variant: 'primary' | 'secondary' | 'ghost' (default: 'primary')
- size: 'sm' | 'md' | 'lg' (default: 'md')
- disabled?: boolean
- children: React.ReactNode
- onClick?: () => void
- type?: 'button' | 'submit'
- className?: string (for extension)

**Styling (Typeform-inspired):**
- Primary: black background, white text, no border radius (or 2px max)
- Secondary: white background, black border, black text
- Ghost: transparent, black text, no border

**Sizes:**
- sm: px-3 py-1.5 text-sm
- md: px-4 py-2 text-base
- lg: px-6 py-3 text-lg

**States:**
- Hover: slight opacity change or background shift
- Disabled: opacity-50, cursor-not-allowed
- Focus: outline using --color-ink

Use CSS variables from globals.css. Use Tailwind classes where possible, inline styles for CSS variables.

Add "use client" directive.
  </action>
