# Hall Head Table Layout

Floor plan for the Reception Hall featuring a 12-foot head table configuration.

## Layout Elements

| Element | Location/Details |
|---------|------------------|
| Sign-in / Gift Table | Near entrance |
| Bar | Along wall |
| Buffet Counter | Service area |
| Cake Table | Designated spot |
| Sweetheart Table | Central front position |
| 12 ft Head Table | Front of room |
