# Wedding Booking Form

> **Use the Form Builder:** This form is available in the Forms section for data collection. This document serves as field reference.

## Purpose

Collect initial contact and event details when a prospective client inquires about wedding services. This information is used for follow-up, vendor coordination, and payment tracking.

## Contact Information Fields

| Field | Description | Required |
|-------|-------------|----------|
| Name | Client's full name | Yes |
| Address | Mailing address | Yes |
| Phone | Primary phone number | Yes |
| Work Phone/Email | Secondary contact method | No |

## Event Details Fields

| Field | Description | Required |
|-------|-------------|----------|
| Ceremony | Ceremony type or style | Yes |
| Guest Count | Expected number of guests | Yes |
| Ceremony Location | Where ceremony will be held | Yes |
| Type of Event | Wedding, reception, etc. | Yes |
| Time | Event start time | Yes |
| Reception Location | Where reception will be held | Yes |
| Coordinator | Assigned coordinator name | No |
| Additional Needs | Special requests or requirements | No |

## Vendor Tracking Fields

| Field | Description | Required |
|-------|-------------|----------|
| After Hours Rental | Rental vendor and date | No |
| Caterer | Catering vendor and date | No |
| Linens | Linen vendor and date | No |
| Planning Appt. | Planning appointment date | No |
| Decor Appt. | Decor appointment date | No |
| 2 Weeks Out Appt. | Final appointment date | No |
| Menu Tasting | Tasting appointment date | No |
| Flowers/Florist | Florist vendor info | No |

## Payment Fields

| Field | Description | Required |
|-------|-------------|----------|
| Payment Due | Amount and due date | Yes |
| Damage Deposit | Deposit amount | Yes |
| Returned | Deposit return status | No |
| Anniversary Email | Client email for anniversary followup | No |
| Payment Tracking | Date, collected by, and amount for each payment | Yes |

## Notes

- Form responses create a client record for tracking
- Vendor information should be updated as vendors are confirmed
- Payment tracking table captures multiple payments over time
- Follow up within 24 hours of initial inquiry
