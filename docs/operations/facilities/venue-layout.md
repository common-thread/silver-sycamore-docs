# Venue Layout Guide

Sample venue layouts for guest counts ranging from 100-200 guests. Maximum indoor capacity at Silver Sycamore is 225 guests with no guest minimum.

## Important Notes

- This document shows sample layouts only
- Couples receive their own editable document during planning
- Winter and summer (or rain day) receptions may take place in Reception Hall or Saloon with different layouts
- Dance floor location is generally consistent unless client chooses otherwise
- A round table is removed in December for the Christmas tree (marked in red on winter layouts)
- Head table type and location can dramatically change the layout
- We do NOT have enough tables for all guest tables to be rectangular, but several can be incorporated
- Gift table, guest book, and seating chart typically go in "entry tables" area
- Sweets table is generally a 6' or 8' rectangle
- Nothing may be placed in front of storage room doors

## Master Layout Reference

Table limits and constraints for event planning:

| Constraint | Limit |
|------------|-------|
| Maximum round guest tables | 24 (fewer with large head table) |
| Maximum rows in center area | 9 (7-8 is ideal) |
| Maximum tables in addition (winter ceremony) | 11 |
| Indoor capacity | 225 guests |

**Do not change the size of any tables in editable versions - they are roughly to scale.**
