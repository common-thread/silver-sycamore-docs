# Sheet 1: _Overview_

| Document Overview - **Please Read** |
| --- |
| Please Note: |
| -- This document shows sample venue layouts for a guest count ranging from 100 - 200 guests. |
| -- Our max capacity at Silver Sycamore indoors is 225 guests. We do not have a guest minimum. |
| -- To view the various layouts by guest count, please navigate using the tabs on this document. |
| -- This document is NOT editable. Couples will be provided with their own editable document in the planning process. |
| -- Winter and Summer (or rain day ) receptions may take place in Reception Hall or Saloon therefore the layout is different. |
| -- Generally (unless unless you choose not to have one), your dance floor will be the same location . |
| -- A round table will be removed in December to make room for the tree. This table is marked in red in the winter layouts. |
| -- Your type and chosen location of your head table can dramatically change the layout. Therefore it is omitted here. |
| -- We do NOT have enough tables for all guest tables to be rectangular, but we do have several, so they can be incorporated into the design. |
| -- Gift table, guest book sign in table, and table seating chart are usually in the area marked as "entry tables." |
| -- The sweets table is generally a 6' or 8' rectangle. |
| -- Nothing may be placed in front of the storage room doors. |
