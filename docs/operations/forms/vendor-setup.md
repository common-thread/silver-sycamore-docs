# Vendor Setup Form

> **Use the Form Builder:** This form is available in the Forms section for data collection. This document serves as field reference.

## Purpose

Coordinate vendor arrivals, setup requirements, and event logistics for the day of the event. This form consolidates all vendor information and setup details for staff reference.

## Event Information Fields

| Field | Description | Required |
|-------|-------------|----------|
| Name | Client name | Yes |
| Date | Event date | Yes |
| Time | Event start time | Yes |
| Guest Count | Number of guests | Yes |
| Wedding Location | Ceremony venue | Yes |
| Reception Location | Reception venue | Yes |

## Setup Details Fields

| Field | Description | Required |
|-------|-------------|----------|
| Chair Count | Number of chairs needed | Yes |
| Guest Tables | Round or Banquet style | Yes |
| Dance Floor | Dance floor setup details | Yes |

## Cake Vendor Fields

| Field | Description | Required |
|-------|-------------|----------|
| Cake Vendor | Bakery name | Yes |
| Drop Off Time & Date | When cake will be delivered | Yes |

## Music Vendor Fields

| Field | Description | Required |
|-------|-------------|----------|
| Music Vendor | DJ or band name | Yes |
| Add Ons | Additional music services | No |

## Linen Vendor Fields

| Field | Description | Required |
|-------|-------------|----------|
| Linen Vendor | Linen company name | Yes |
| Drop Off Time & Date | When linens will be delivered | Yes |
| Setup Location | Where linens should be placed | Yes |

## Security Fields

| Field | Description | Required |
|-------|-------------|----------|
| Security | Security vendor or details | No |

## Other Rentals Fields

| Field | Description | Required |
|-------|-------------|----------|
| Other Rentals | Additional rental items and vendors | No |

## Bar Fields

| Field | Description | Required |
|-------|-------------|----------|
| Alcohol Type | Type of bar service and selections | Yes |
| Ordered | Alcohol order status | Yes |
| Bartender | Assigned bartender | Yes |

## Notes

- Use this form for day-of setup coordination
- All vendors should have confirmed arrival times
- Review this form during pre-event staff meeting
- Ensure all rental items are accounted for at delivery
