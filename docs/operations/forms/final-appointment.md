# Final Appointment Form

> **Use the Form Builder:** This form is available in the Forms section for data collection. This document serves as field reference.

## Purpose

Comprehensive form for the final pre-wedding appointment, typically held two weeks before the event. Captures all remaining details including vendor coordination, timeline, and setup requirements.

## Basic Event Information

| Field | Description | Required |
|-------|-------------|----------|
| Meeting Date | Date of final appointment | Yes |
| Name | Client name | Yes |
| Date of Event | Wedding date | Yes |
| Time | Event start time | Yes |
| Guest Count Guaranty | Final confirmed guest count | Yes |
| Wedding Site Location | Ceremony venue | Yes |
| Reception Location | Reception venue | Yes |

## Notes
