# Decor Appointment Form

> **Use the Form Builder:** This form is available in the Forms section for data collection. This document serves as field reference.

## Purpose

Capture decor preferences and decisions during the decor planning appointment with clients. This form documents decoration choices for the ceremony, guest tables, sweetheart table, cake table, and additional decor elements.

## Client Information Fields

| Field | Description | Required |
|-------|-------------|----------|
| Name | Client's name | Yes |
| Event Date | Date of the event | Yes |
| Date | Date of the decor appointment | Yes |

## Ceremony Decor Fields

| Field | Description | Required |
|-------|-------------|----------|
| Draping Lights | On or Off preference | Yes |
| Ceremony Decor Notes | Additional ceremony decoration notes | No |

## Guest Table Fields

| Field | Description | Required |
|-------|-------------|----------|
| Guest Table Centerpieces | Centerpiece style and details | Yes |

## Sweetheart Table Fields

| Field | Description | Required |
|-------|-------------|----------|
| Sweetheart Table Decor | Decoration style and details | Yes |

## Cake Table Fields

| Field | Description | Required |
|-------|-------------|----------|
| Cake Table Decor | Table decoration details | No |
| Cake Cutting Set | Silver Sycamore (SS) or Client's own (Theirs) | Yes |
| Champagne Glasses | Silver Sycamore (SS) or Client's own (Theirs) | Yes |

## Additional Fields

| Field | Description | Required |
|-------|-------------|----------|
| Additional Decor | Any other decoration requests or notes | No |

## Notes

- Complete this form during the decor appointment meeting
- Document all client preferences for setup day reference
- Confirm which items client is providing vs. venue-provided
- SS indicates Silver Sycamore-provided items
