# Pine Street Cafe Recipe App

## Status: Live (Piloting)

## Links

- **Live App**: https://pine-street-cafe-recipes.vercel.app
- **Recipe Count**: 278 recipes
- **Source Data**: Convex database (`recipes` table)

## Description

Web application for Pine Street Cafe kitchen staff to search and access recipes. Features include:

- Search by recipe name
- Filter by category (9 categories)
- Mobile-friendly interface
- Real-time data from Convex backend

## Source Information

Recipes were extracted from the Pine Street Cafe physical recipe binder and digitized into the Convex database during the January 2026 engagement.
